# Student Management System (C++)

This is a beginner-friendly console-based project built to manage student records efficiently using file handling.

## Features
- Add student records
- Display all students
- Update student details
- Delete student records
- Data is stored permanently using file handling

## How it Works
The program uses a menu-driven system where users can choose operations. All data is stored in a file, so records are even after closing the program.

## Tech Used
- C++
- File Handling
- Structured Programming
